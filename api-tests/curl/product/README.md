# CRUD de Produtos – ServeRest

Base URL:

```text
https://serverest.dev
```

---

## 🎯 Fluxo ideal para seu portfólio

1. Criar produto
2. Listar produtos
3. Buscar por ID
4. Atualizar produto
5. Excluir produto
6. Validar cenários negativos

---

## 💡 Dica importante de QA

Para produtos, sempre valide:

* status code
* campos obrigatórios
* tipos de dados (preço não pode ser string)
* quantidade >= 0
* duplicidade de nome
